# MultiLearn

A web application developed during the FrauenLoop Full-Stack Web Development Program.

MultiLearn was created to practice full-stack web development concepts by combining frontend and backend technologies in a real-world project. The application integrates geolocation services and follows a structured Flask architecture.

## Technologies

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Jinja2
- SQLite
- OpenStreetMap API
- Git & GitHub

## Features

- Interactive web interface
- Flask application structure
- Template rendering with Jinja2
- Database integration
- Geolocation using OpenStreetMap API
- Responsive design

## What I Learned

During this project I practiced:

- Building a Flask application
- Organizing a full-stack project structure
- Working with templates and routing
- Integrating external APIs
- Handling database interactions
- Frontend and backend integration
- Version control using Git and GitHub
